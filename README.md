# 📚 Library Management System (Perpustakaan Kampus)

## 📖 Description

This project is a web-based Library Management System built using PHP and MySQL.
It helps manage books, members, transactions, and reports in a simple and efficient way.

## 🚀 Features

* User Authentication (Login & Register with CAPTCHA)
* Book Management (Add, Edit, Delete)
* Member Management
* Borrowing & Returning System
* Reports with tables and charts
* Print report (table only)

## 🛠️ Technologies Used

* PHP (Native)
* MySQL
* Bootstrap 5
* JavaScript

## 📂 Project Structure

* `index.php` → Main homepage
* `dashboard.php` → Admin dashboard
* `laporan.php` → Reports page
* `auth/` → Login & Register system
* `config/` → Database connection
* `img/` → Images & assets

## ⚙️ Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/Rizki-Code156/perpustakaan-php.git
   ```
2. Move the folder to `htdocs` (XAMPP)
3. Import the database to phpMyAdmin
4. Run via browser:

   ```
   http://localhost/perpustakaan-php
   ```
